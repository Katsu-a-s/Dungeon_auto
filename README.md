# Dungeon Auto

Python + [pygame](https://www.pyga.me/) 製のローグライク風ダンジョン探索ゲーム。
メインファイルは **`dungeon_rev121.py`**(単一ファイル構成、約6,700行)。

> このリポジトリは Claude によるルーティン作業(1時間ごとの自動開発)で
> 少しずつ機能追加・改善が行われています。このREADMEも実装状況に合わせて
> 逐次更新されます。

## 遊び方

```bash
pip install pygame
python3 dungeon_rev121.py
```

- 画面サイズ 880x720、キーボード + マウス操作対応。
- セーブデータ等(`savefile*.json` / `autosave.json` / `settings.json` など)は
  実行時にリポジトリ直下へ自動生成されます(`.gitignore` 済みでコミット対象外)。

## 操作方法

ゲーム内タイトル画面右上の **`[F1] Help`** ボタン(または `F1` キー)から、
いつでもコントロール一覧を確認できます。

| 場面 | キー | 内容 |
|---|---|---|
| タイトル | 矢印キー / `Space` | メニュー選択 / ゲーム開始 |
| タイトル | `T` | 難易度切り替え(Easy/Normal/Hard) |
| タイトル | `N` | キャラクター選択 |
| タイトル | `G` | ゲームデータ(ロード/コンティニュー/設定) |
| タイトル | `R` | 記録(実績/統計/図鑑/エコーバトル/デイリーランキング) |
| タイトル | `Y` | デイリーチャレンジ開始 |
| タイトル | `H` | 隠しステージ(全クリア後に出現) |
| タイトル | `F1` | ヘルプ画面 |
| ダンジョン探索 | 矢印キー | 移動 |
| ダンジョン探索 | `Q` | セーブメニュー |
| ダンジョン探索 | `K` | スキルツリー |
| ダンジョン探索 | `I` | ポーション使用 |
| バトル | `A`/`P`/`B`/`R`/`D`/`F` | 攻撃/ポーション/爆炎石/逃走/防御/集中 |
| バトル | `Up`/`Down`, `Space`/`Enter` | コマンド選択/決定 |
| 共通 | `Esc` | 戻る・キャンセル |
| 共通 | `M` | 全音声(BGM/SE)をミュート/解除(NEW) |

## 現在の実装状況

### ダンジョン・進行
- 全 **3ステージ × 30階 = 90階**(`STAGE_LENGTH`/`STAGE_COUNT`)+ 全クリア後の隠しステージ(ボス戦)。
- 迷路生成 + フロア修飾(霧/豊穣/岩場/煌めき/静穏/幸運/共鳴など、`FLOOR_MODIFIERS`、計12種類、NEW: Resonant Floor)。
- ショートカット移動・キー長押しでの連続移動、氷床での自動スライドなど操作性の作り込み。
- 20種類以上の特別部屋(モンスターの巣・祠・裂け目・賭博場・キメラ部屋・血月イベント等)を
  フロアごとに抽選配置。

### バトル
- ターン制バトル(攻撃/ポーション/爆炎石/逃走/防御/集中)+ コンボ倍率システム。
- 毒/呪い/仲間バフ/トーテムバフなどの状態異常・バフ。
- 通常敵17種(フロア帯で出現テーブルが変化)+ エリート個体(出現率12%)。
- 10階ごとにボス出現、90階超の隠しステージにも専用ボス。
- クリティカルヒット(スキル「会心の一撃」で発生率上昇、Fortunate Floorでは
  スキル未取得でもそのフロア限定でクリティカルが発生するように)。
- ボスはHPが50%を切ると「フェーズ2(激怒)」に移行し、攻撃力が上昇(**NEW**:
  HPバーが脈打つ赤に変化し `* ENRAGED *` 表示が戦闘終了まで持続するようになり、
  激怒状態が一目でわかるように)。

### キャラクター・成長
- 4種の主人公タイプ(Warrior/Guardian/Scholar/Scout)でプレイスタイルを選択可能。
- レベル/経験値/所持品(ポーション・爆炎石・食料など)。
- 5系統×3段+奥義1つ、計16スキルのスキルツリー。
- 難易度3段階(Easy/Normal/Hard)でパラメータ調整可能。

### やり込み要素
- 実績システム(36種)+ 称号システム(実績達成で二つ名を獲得)。
- 図鑑(Bestiary):遭遇したモンスター/ボス/発見アイテムを記録・詳細表示。
- プレイ統計画面、セーブ3スロット+オートセーブ、デイリーチャレンジ(共通シード+ランキング)。
  統計には通算撃破数・宝箱開封数などに加え、**通算で到達した最深フロア(NEW)**も記録。
- **エコーバトル**:これまで倒したボスをフロア別に再挑戦できるモード。

### UI/UX
- タイトル画面はマウスクリック操作にも対応した階層メニュー(つやのあるボタンUI)。
- BGM/SE音量を個別調整できる設定画面 + `M`キーでのワンタッチ全ミュート切り替え。
- 実績・統計一覧のスクロール対応、図鑑詳細のクリックプレビュー。
- **ヘルプ画面(`[F1]`)**:操作方法をタイトル/探索/バトル別にまとめて確認可能。
- **低HP警告**:残りHPが最大値の20%を切ると、探索中・バトル中を問わず
  画面端が心拍のように速く脈打つ赤い縁取りで警告(崩落演出とは脈拍の速さで区別)。
- **クリティカル/コンボフィニッシャー演出**:クリティカルヒットやコンボ
  フィニッシャーが発動した瞬間、画面全体が短く光るフラッシュ演出を追加し、
  攻撃の爽快感を強化(フィニッシャーはより長く・ピンクがかった色で発光)。
- **ボス階警告**:未撃破のボスがいるフロアに足を踏み入れると、
  「Welcome to floor」表示と一緒に脈打つ赤文字で警告メッセージが表示され、
  階段に着く前にポーションや爆炎石の準備をする猶予が生まれるように。
- **画面シェイク演出**:被弾時・クリティカルヒット・コンボフィニッシャー発動時に
  画面が数フレームだけ小刻みに揺れるようになり、打撃の重みを強化(ボスの
  フェーズ2/激怒中に被弾すると揺れが一段大きくなる)。
- **ダメージポップアップ**:攻撃がヒットした瞬間、命中した対象(敵/プレイヤー)の
  頭上にダメージ数値がふわっと浮かび上がってフェードアウトするように。固定の
  メッセージ欄だけでは伝わりにくかった「どこで何が起きたか」を視覚的に補強し、
  クリティカルヒットは金色・コンボフィニッシャーはピンク色で一回り大きく強調表示。
- **実績解除トースト**:実績を解除した瞬間、画面上部にバッジ画像付きの
  ゴールドのバナーが上からスライドインして「ACHIEVEMENT UNLOCKED!」と実績名を
  表示するように。従来は他の雑多な通知と同じ地味な1行テキストで、探索中/バトル中
  問わず気づきにくかったが、専用の演出にしたことで達成感がひと目で伝わるように
  なった(探索・バトル・メニューなどどの画面でも表示される)。

### サウンド・アセット
- `image/`:敵・ボス・アイテム・床・エフェクトなど201点のスプライト。
- `sound/`:ステージ別BGM(タイトル/フィールド/バトル、全ステージ分)+ SE + ジングル(レベルアップ/勝利/ゲームオーバー)。
- **実績解除ジングル(NEW)**:実績解除トーストが表示される瞬間、レベルアップ
  ジングルを再生して音でも達成に気づけるように(従来はゴールドバナーの
  スライドインのみで、BGM/SEに気を取られていると見逃すことがあった)。
  ミュート設定・SE音量設定にも追従する。

## 直近の更新履歴(最新順)

- 実績解除トーストの表示と同時にレベルアップジングルを再生するように変更。
  従来はゴールドバナーのスライドインという視覚演出のみだったため、探索中や
  バトル中で画面に集中していないと見逃すことがあり、聴覚でも気づけるようにした。
- 実績解除時の通知を、他のメッセージと同じ地味な1行テキストから、バッジ画像付きの
  専用ゴールドバナー(画面上部にスライドイン+パルスする縁取り)に変更。達成の
  瞬間がひと目で分かるようにし、ご褒美感を強化した。
- プレイ統計画面に「通算で到達した最深フロア」を追加。従来はゲームオーバー画面で
  そのランの最深到達フロア(`fl_max`)が一度表示されるだけで、歴代記録は残って
  いなかったため、他の生涯統計と同様に永続化して確認できるようにした。
- 攻撃がヒットした瞬間、対象の頭上にダメージ数値が浮かび上がってフェードアウトする
  ダメージポップアップ演出を追加(クリティカルヒット/コンボフィニッシャー時は
  金色・ピンク色で一回り大きく強調)
- 被弾/クリティカルヒット/コンボフィニッシャー発動時に画面が数フレーム揺れる
  画面シェイク演出を追加(ボスのフェーズ2/激怒中の被弾はより大きく揺れる)
- ボス階(未撃破)に入った瞬間、脈打つ赤文字で警告メッセージを表示するように
  なり、不意打ちされずに準備してから戦えるように改善
- 新しいフロア特性「Resonant Floor(共鳴の床)」を追加(コンボを積むごとの
  ダメージボーナスが通常より大きく伸びる:1段あたり+10%→+15%、上限+50%→+75%)
- ボスのフェーズ2(激怒)発動中、HPバーが脈打つ赤に変化し `* ENRAGED *` 表示が
  戦闘終了まで持続するように変更(従来は発動時の一度きりのメッセージのみだった)
- 新しいフロア特性「Fortunate Floor(幸運の床)」を追加(スキル未取得でもそのフロアではクリティカルヒットが発生するようになる)
- クリティカルヒット/コンボフィニッシャー発動時に画面全体が光るフラッシュ演出を追加
- 残りHPが20%を切ると画面端が脈打つ赤で警告する低HP警告演出を追加(探索・バトル共通)

## 開発方針

- 既存のコードスタイル・命名規則を踏襲。
- 1回の自動実行あたりの変更は小〜中規模に留め、既存のゲームプレイを壊さない。
- 変更のたびに構文チェック・簡易な動作確認(ヘッドレス起動テスト等)を実施。
- メインファイルを変更するたびに `dungeon_revNNN.py` のNNNを1つ上げた
  新しいファイル名で保存し(例: `dungeon_rev120.py` → `dungeon_rev121.py`)、
  バージョンごとの変更履歴をファイル名からも追えるようにする。
